# 🧠 Logical Programs

> Concepts are clear — now it's time to sharpen the logic!

This folder contains intermediate to pro-level C programs focused on
building strong logical thinking and problem-solving skills.

## 💡 What's Inside?
Programs based on:
- Number logic (Prime, Fibonacci, Armstrong, Palindrome)
- Pattern printing (Pyramids, Stars, Diamonds)
- Sorting & Searching algorithms (Bubble, Selection, Binary Search)
- Recursion & Dynamic Programming
- Mathematical problems

## 🎯 Goal
To move beyond concepts and write clean, efficient, logical code
that solves real problems — one program at a time.

## 📈 Level
Intermediate → Advanced
